Installation : Windows

  1. Click 'download.zip' for the latest release {% if page.githublinks.title %}({{ page.githublinks.title }}){% endif %} to download Defunct.

  2. Unzip the folder 'defunct' to a memorable location.

  3. In the Command Prompt, cd to the unzipped location. For example:

    cd "<unzipped location>\defunct\"

  4. You can now run 'defunct.exe' with the following command, which will then show you valid arguments you can use.

    defunct

    To run a script, pass in the file path as a parameter.

    defunct "examples/example01.txt"
